# Système Photovoltaïque Intelligent à Suivi Solaire

## Description
Ce mini-projet consiste en la réalisation d’un système photovoltaïque intelligent capable de suivre automatiquement la direction du soleil pour optimiser la production d’énergie électrique. Le système surveille également plusieurs paramètres électriques et environnementaux.

## Architecture
- **Arduino UNO** : contrôle le mouvement du panneau solaire  
- **ESP8266** : collecte et transmet les données des capteurs  

## Capteurs
- Luminosité  
- Courant électrique  
- Température et humidité  
- Poussière  

## Affichage
Les données sont transmises en temps réel sur un **smartphone** via l’application **Blynk**, permettant une supervision simple et efficace.

## Matériel
Arduino UNO, ESP8266, panneaux solaires, moteurs, capteurs variés.
